 This is optiboot v6.2 bootloader with additions for the Duino Mega32 board from Conrad.
 The project is build with Atmel Studio 7, project files included
 
 It's compatible to the Arduino IDE and is tested with the version 1.6.11 under Win10
 For use it with the Windows version, copy the Duino_Mega32 directory to 
 C:\Users\%username%\Documents\Arduino\hardware and choose the board 
 "Duino_Mega32-optiboot512".
